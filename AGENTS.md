## Tech Stack
- Frontend: React 18 + TypeScript
- Backend: Express.js + Node.js
- Database: MongoDB + Mongoose
- Testing: Jest + React Testing Library
- Styling: Tailwind CSS

## Coding Standards
- ESLint + Prettier 사용
- 함수명은 camelCase
- 컴포넌트명은 PascalCase
- API 엔드포인트는 RESTful 방식
- 모든 함수에 JSDoc 주석 필수

## Project Structure
- src/
- components/     # 재사용 가능한 컴포넌트
- pages/         # 페이지 컴포넌트
- hooks/         # 커스텀 훅
- api/           # API 유틸리티
- utils/         # 헬퍼 함수

## Environment
- Node.js 18+
- MongoDB 6.0+
- 포트: 3000 (프론트), 5000 (백엔드)
